What This Document Covers  This is a companion document to the iCM Voice Discovery Agent Complete Build Guide v1.1. It covers three additions that work alongside the agent you have already built:  •   The SI Manager Dashboard — a separate internal web application at admin.icaremanager.com protected by Google SSO, where SI Managers add new customers, generate discovery links, and track session completion progress  •   The Consent System — legal consent language added to the pre-session form on the agent page, plus a consent log that records who consented, when, and from which IP address  •   The n8n Updates — two new webhook workflows that support the dashboard and the consent log  Build These in This Order  Step 1: Update the agent web page to add the consent form (Phase 2 of this document)  Step 2: Update n8n to capture consent data and write it to the Google Sheet (Phase 3)  Step 3: Build the SI Manager Dashboard in AntiGravity (Phase 4)  Step 4: Connect the dashboard to the Google Sheet and test everything end to end (Phase 5)  The agent itself (Vapi, Claude prompt, ElevenLabs) does not change in this document.  2.
